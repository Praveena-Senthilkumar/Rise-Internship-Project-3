# Rise Internship Project 3
# Serverless Contact Form using AWS Lambda

## Overview
This project is a simple **serverless contact form application** built using AWS cloud services. The application allows users to submit their name, email, and message through a web form, and the data is stored securely in a cloud database.

The frontend is developed using **HTML and JavaScript**, while the backend is powered by AWS serverless services. The form sends data to an API endpoint, which triggers a Lambda function to process and store the data.

## Technologies Used
- AWS Lambda
- Amazon API Gateway
- Amazon DynamoDB
- HTML
- JavaScript

## Architecture
User submits the contact form through the browser. The request is sent to **Amazon API Gateway**, which triggers an **AWS Lambda function**. The Lambda function processes the request and stores the submitted data in **Amazon DynamoDB**.

## Features
- Serverless backend architecture
- Contact form data storage in cloud database
- REST API integration
- Simple and responsive frontend form
- Fully managed AWS infrastructure

## Summary
This project demonstrates how to build a **serverless web application** using AWS services. It highlights the integration of API Gateway, Lambda, and DynamoDB to create a scalable backend without managing servers. The project showcases basic cloud architecture and API-based data handling for web applications.
